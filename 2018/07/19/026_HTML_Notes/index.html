HTML 学习笔记

1. HTML 基础

HTML 速查表

1.1 <html> 标签

  • 一张网页有且仅有一个 <html> 标签;
  • 浏览器能解析 <html> 标签之外的内容, 但不建议这么做;

1.2 <body> 标签

  • 存放网页上的可见元素;

1.3 <head> 标签

  • 存放网页上的不可见元素;

1.4 <title> 标签

  • 搜索引擎把 title 作为搜索结果的标题;
  • 浏览器在标签页显示 title ;

1.5 <h1-h6> 标签

  • 可以嵌套子元素;

1.6 <p> 标签

  • paragraph;
  • 可以嵌套子元素;
  • 嵌套 <p> 标签在网页上不会显示效果, 但层级分明, 便于添加样式;

1.7 <div> 标签

  • division;
  • 非常干净的盒子, 唯一的功能就是用于容纳其它元素;

1.8 <a> 标签

  • anchor;
  • 锚链接;

1.8.1 href 属性

  • hypertext reference;
  • 用于指定超链接目标的 URL;

1.8.2 target 属性

  • _self : 默认值; 在当前页面打开;
  • _blank : 在新标签页打开;

1.9 <img> 标签

  • 单标签, 结构树的最后一个节点, 不能含有子元素;
  • 所以它不是一个容器;
  • 所以也不需要结束标签;

1.9.1 src 属性

  • 网络路径;
  • 绝对路径;
  • 相对路径;

1.9.2 alt 属性

  • alternate;
  • 当图片无法显示时, 显示替代的文字;

1.10 <table> 标签

1.10.1 <tr> 标签

  • table row;

1.10.2 <td> 标签

  • table data cell;

1.10.3 <th> 标签

  • table head cell;

1.10.4 <thead> 标签

  • 容器; 用来放 <th> 标签; 语法更严谨;

1.10.5 <tbody> 标签

  • 容器; 用来放 <td> 标签; 语法更严谨;

1.10.6 border 属性

1.11 <header> 标签

  • 页眉;
  • 重要而抽象的信息, 一般为目录;
  • 指导搜索引擎按优先级抓取网页;
  • 在功能上并没有比 <div> 更多, 只是有语义, 更便于人或爬虫读代码;

1.12 <footer> 标签

  • 页脚;

  • 重要但不常用的信息, 一般为链接;

  • 在功能上并没有比 <div> 更多, 只是有语义, 更便于人或爬虫读代码;

1.13 <hr> 标签

  • horizonal rule; 水平分割线;
  • 单标签;

1.14 <link> 标签

  • 单标签;
  • 既可以放在 <head> 里, 也可以放在 <body> 里;
  • 一般放在 <head> 里;
  • 一般用于引入 .css 文件;
  • href 属性;
  • ref 属性: 一般为 stylesheet.

1.15 <script> 标签

  • 既可以放在 <head> 里, 也可以放在 <body> 里;
  • 一般放在 <body> 的最后, 可以先显示静态内容;
  • 如果页面对 js 非常依赖, 就把它放在 <head> 里;
  • 用于引入 js;
  • js 可以写在标签里, 也可以写在 .js 文件里;

1.16 <button> 标签

  • 一般放在<form>里;
  • 配合 js;

1.17 <abbr> 标签

  • abbreviation; 缩写;
  • 行内元素;
  • title 属性: 相当于注释; 鼠标悬停会显示;
  • 换成 <p> 或者其它标签也能鼠标悬停显示原意;
  • 但一般专门用 <abbr> 来表示;
  • 有的浏览器会为<abbr> 添加一些样式, 有时是在下面加虚线, 有时是改变字体;

1.18 <code> 标签

  • 行内元素;
  • 放少量代码;
  • 在 <head> 中用 <style> 添加样式会非常方便, 不用专门去指定一个类;

1.19 <pre> 标签

  • 块级元素;
  • 放大量代码;
  • 在 <head> 中用 <style> 添加样式会非常方便, 不用专门去指定一个类;
