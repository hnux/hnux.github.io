德国史纲

发表于 2019-10-13 | 分类于 历史 | 阅读次数:

德国性格: 逻辑 -> 推理 -> 法治 -> 不讲人情

德意志第一帝国

  • 即神圣罗马帝国.
  • 962 - 1806.

渊源

  • 罗马共和国, BC509 - BC27.
  • BC27, 屋大维建立罗马帝国.
  • 293年, 戴克里先采用「四帝共治」.
  • 395年, 狄奥多西一世将帝国分给两位儿子, 从此分裂为东罗马帝国(即拜占庭帝国,1453年被奥斯曼帝国消灭, 改君士坦丁堡为伊斯坦布尔)和西罗马帝国(定都拉文纳, 外患不断, 476年解体).
  • 西罗马帝国被法兰克帝国(486年, 克洛维在巴黎建立墨洛温王朝; 843年, 路易一世的三个儿子订立凡尔登条约, 将帝国一分为三, 即后来法德意的雏形)继承.
  • 东法兰克王国(843-962)

始于奥托王朝

  • 962年, 东法兰克国王奥托一世在罗马由教皇约翰十二世加冕称帝.
  • 973年, 成为罗马的监护人与皇帝和罗马天主教世界的最高统治者.

大选侯时期

  • 七大选帝侯: 巴伐利亚, 奥地利, 普鲁士, 萨克森, 匈牙利, 特里尔大主教, 科隆大主教.
  • 1257年, 首次选举国王时候产生.
  • 1273年, 奥地利的鲁道夫一世建立哈布斯堡王朝.

终于拿破仑战争

  • 1804年5月18日, 拿破仑称帝, 弗朗茨二世纠合第三次反法同盟.
  • 1805年12月16日, 拿破仑打赢「三皇会战」, 逼迫弗朗茨二世签订「普雷斯堡和约」.
  • 1806年7月12日, 在拿破仑的威逼利诱下, 16个神圣罗马帝国的成员邦签订了「莱茵邦联条约」, 脱离帝国, 加入邦联.
  • 1806年8月6日, 弗朗茨二世放弃神圣罗马帝号, 仅保留奥地利帝号,神圣罗马帝国名存实亡.
  • 1871年, 俾斯麦统一德意志, 神圣罗马帝国正式灭亡.

德意志邦联

  • 拿破仑征服德意志后将300多个王国合并为30多个, 莱茵兰左岸划归法国, 保留了奥地利(天主教)和普鲁士(新教), 奥普之间20多个王国组成法帝国的附庸「莱茵邦联」.
  • 拿破仑战争(1803-1815)结束后, 英俄奥普等战胜国构建维也纳体系.
  • 维也纳体系把德意志地区的34个邦国和自由城市组成德意志邦联(邦联: 各有主权 ; 联邦: 主权合一).

普鲁士

  • 1834年, 占据了黄金海岸线的普鲁士构建德意志关税联盟, 唯独排斥内陆国家奥地利.

  • 俾斯麦孤立化(非国际问题)逐步统一德国.

    • 1864年, 联合奥地利打败丹麦, 收复石勒苏益格和荷尔斯泰因.

    • 1865年8月14日, 普鲁士与奥地利签署加斯泰因条约, 分别管理石勒苏益格和荷尔斯泰因.

    • 1866年, 联合意大利打败奥地利但不吞并, 选择小德意志方案, 德奥分家从此始.

    • 1870年9月2日, 拿破仑三世投降.

    • 871年1月18日, 普鲁士国王威廉一世在凡尔赛宫加冕为帝, 成立德意志第二帝国.

德意志第二帝国

  • 在外交上, 俾斯麦深刻地意识到德国的风险在于周边列强的联合遏杀, 所以在国家利益上始终有所取舍, 他致力于拆分各国的友谊.

    相比之下, 威廉二世则以大国崛起自居, 挑战维也纳体系, 在摩洛哥问题上压制法国, 在波斯尼亚问题上打压俄国, 在海军问题上打压英国, 结果驱使英法俄三国联合扼杀德国.

  • 1914年6月28日, 奥匈帝国皇储被杀;

    1914年7月28日, 奥匈帝国在德国支持下向萨尔维亚宣战, 日耳曼民族掀起一战;

  • 1918年11月3日, 基尔水军起义.

  • 1918年11月9日, 德皇威廉二世逃往荷兰, 德意志第二帝国被社会民主党推翻.

魏玛共和国

  • 1918 - 1933.
  • 极端的民主.
  • 1923年11月9日, 1美元兑4.2万亿马克, 希特勒啤酒馆暴动.

德意志第三帝国

  • 即纳粹德国.
  • 极端的专制.
  • 希特勒选择大德意志方案.
  • 1938年, 德国合并奥地利.
  • 1938年11月9日, 希特勒授意戈培尔发动「水晶之夜」事件, 标志着纳粹对犹太人有组织的屠杀的开始.
  • 1939年, 希特勒联合闪击波兰, 日耳曼民族掀起二战.
  • 二战(1939-1945)失败后, 德意志第三帝国垮台.

东德与西德

  • 1945年5月8日, 德国被一分为四, 由美苏英法管制.
  • 1949年5月23日, 美英法占领区合并成立德意志联邦共和国, 即西德.
  • 1949年10月7日, 苏占区成立德意志民主共和国, 即东德.
  • 1961年8月13日, 东德始建柏林墙, 阻止东德居民逃往西柏林.
  • 1989年11月9日, 柏林墙倒塌.

两德统一

  • 1990年10月3日, 东德并入西德, 打破了德国历史上「民主代表分裂, 专制代表统一」的魔咒. 德国总理科尔声明「德国的领土野心已经满足」, 让邻居放心.
  • 1991年12月26日, 苏联解体, 从英伦三岛到乌拉尔山, 所有国家的政治制度都统一为议会民主制.
